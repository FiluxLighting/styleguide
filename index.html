Sistema Tipográfico

Una sola familia tipográfica: Inter. Moderna, legible y optimizada para web. Reduce el tiempo de carga y mantiene consistencia visual.

H1 - Título Principal (Light)

H2 - Título de Sección (Semibold)

H3 - Subtítulo (Medium)

H4 - Apartado (Medium)

Texto lead - Para introducciones importantes

Párrafo regular - Para contenido principal con buena legibilidad

Texto pequeño - Para detalles y especificaciones
