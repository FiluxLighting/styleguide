Filosofía de Diseño

Minimalismo

Espacios en blanco generosos, elementos esenciales, sin distracciones para destacar los productos.

Funcionalidad

Diseño centrado en el usuario, navegación intuitiva, conversión optimizada sin sacrificar la estética.

Sistema Tipográfico

Una sola familia tipográfica: Inter. Moderna, legible y optimizada para web. Reduce el tiempo de carga y mantiene consistencia visual.

H1 - Título Principal (Light)

H2 - Título de Sección (Semibold)

H3 - Subtítulo (Medium)

H4 - Apartado (Medium)

Texto lead - Para introducciones importantes

Párrafo regular - Para contenido principal con buena legibilidad

Texto pequeño - Para detalles y especificaciones

/* Jerarquía tipográfica */ h1 { font-size: var(--text-5xl); font-weight: var(--font-light); } h2 { font-size: var(--text-3xl); font-weight: var(--font-semibold); } .text-lead { font-size: var(--text-lg); font-weight: var(--font-light); } p { font-size: var(--text-base); color: var(--color-text-secondary); }
